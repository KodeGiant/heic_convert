<task type="auto">
  <name>Task 2: Implement HEIC conversion and downloads</name>
  <files>app.js</files>
  <action>
Add to app.js (continuing from Task 1):

**State additions:**
```js
let convertedBlobs = []; // Array of { name: string, blob: Blob }
```

**Convert button handler:**
- Get #convert-btn; add 'click' listener
- On click:
  1. Disable convert-btn, show #status (remove hidden), set status text to "Converting..."
  2. Set convertedBlobs = []
  3. Clear #results-list innerHTML; hide #results (add hidden attribute)
  4. Get quality from #quality-slider as float: `const quality = parseInt(slider.value) / 100`
  5. Process files sequentially with a for...of loop and await (not Promise.all — avoids memory spikes on large batches):
     ```js
     for (const file of selectedFiles) {
       try {
         const blob = await heic2any({ blob: file, toType: 'image/jpeg', quality });
         const name = file.name.replace(/\.heic$/i, '.jpg');
         convertedBlobs.push({ name, blob });
         // Append result item to #results-list (see below)
       } catch (err) {
         // append an error item to #results-list noting file.name failed
       }
     }
     ```
  6. After loop: re-enable convert-btn, hide #status (add hidden), reveal #results (remove hidden) if convertedBlobs.length > 0

**Result item injection (DL-01):**
For each successful conversion, create and append to #results-list:
```html
<li class="result-item">
  <span>{name}</span>
  <a href="{objectURL}" download="{name}">Download</a>
</li>
```
Use `URL.createObjectURL(blob)` for the href. Store the URL so it can be revoked later (or accept minor leak for simplicity — this is a personal tool).

**ZIP download handler (DL-02):**
- Get #download-all-btn; add 'click' listener
- On click:
  1. If convertedBlobs is empty, return
  2. Set button text to "Zipping..."
  3. `const zip = new JSZip()`
  4. For each `{ name, blob }` in convertedBlobs: `zip.file(name, blob)`
  5. `const zipBlob = await zip.generateAsync({ type: 'blob' })`
  6. Create a temporary `<a>` with `href = URL.createObjectURL(zipBlob)` and `download = 'converted.zip'`, click it, then remove it
  7. Restore button text to "Download all as ZIP"

**Error display:**
- If a file fails, append to #results-list:
  ```html
  <li class="result-item"><span>{file.name} — failed</span></li>
  ```
- If ALL files fail, show #status with class 'error' and text "Conversion failed. Ensure files are valid HEIC images."
